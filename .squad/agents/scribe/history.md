## Core Context

- RecipeBoss is an AI-assisted personal recipe library that imports recipes from public URLs, normalizes them with Azure AI Foundry, stores images in Azure Blob Storage, and persists user data in Cosmos DB.
- Frontend stack: React 18, TypeScript, Vite, shadcn/ui, Tailwind CSS.
- Backend and platform stack: ASP.NET Core 10 minimal APIs, Microsoft Entra External ID, .NET Aspire, Azure Static Web Apps, Azure Container Apps.
